== Description ==

TL;DR Pro is a powerful WordPress plugin that automatically generates AI-powered summaries for your content. It helps reduce bounce rates by providing instant, scannable summaries that keep readers engaged.

= Key Features =

* **Multiple AI Providers**: Support for DeepSeek, Google Gemini, Claude Anthropic, and OpenAI GPT
* **Smart Summaries**: Generate concise, engaging summaries that capture the essence of your content
* **Customizable Display**: Choose from 11 color styles and multiple positioning options
* **Bulk Operations**: Generate summaries for multiple posts at once
* **Performance Optimized**: Built-in caching and efficient API usage
* **Multilingual Support**: Works with content in multiple languages
* **Responsive Design**: Beautiful on all devices
* **Accessibility Ready**: WCAG 2.1 compliant

= Supported AI Providers =

* **DeepSeek**: Most cost-effective option with excellent quality
* **Google Gemini**: Free tier available with fast response times
* **Claude Anthropic**: Advanced reasoning capabilities
* **OpenAI GPT**: Industry-standard with multiple model options

= How It Works =

1. Install and activate the plugin
2. Configure your preferred AI provider with API credentials
3. Customize display settings to match your theme
4. Generate summaries manually or automatically
5. Summaries appear as beautiful, interactive buttons on your content

== Installation ==

1. Upload the `tldr-pro` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to TL;DR Pro settings to configure your AI provider
4. Add your API credentials for your chosen provider
5. Start generating summaries!

== Frequently Asked Questions ==

= Do I need to pay for AI API access? =

It depends on the provider you choose:
- **Google Gemini**: Offers a free tier with 60 requests per minute
- **DeepSeek**: Very affordable pay-as-you-go pricing
- **Claude & OpenAI**: Require paid API access

= Can I customize how summaries look? =

Yes! The plugin includes:
- 11 pre-designed color styles
- Custom color options
- Multiple button positions
- Responsive modal display

= Will this slow down my site? =

No. TL;DR Pro is optimized for performance with:
- Efficient caching system
- Lazy loading
- Optimized database queries
- Minimal frontend assets

= Is it compatible with my theme? =

TL;DR Pro is designed to work with any WordPress theme. It includes compatibility styles to prevent conflicts with popular themes and page builders.

== Privacy Policy ==

This plugin sends content to third-party AI services (based on your configuration) to generate summaries. Please review the privacy policies of your chosen AI provider:

* DeepSeek: https://www.deepseek.com/privacy
* Google Gemini: https://policies.google.com/privacy
* Anthropic Claude: https://www.anthropic.com/privacy
* OpenAI: https://openai.com/privacy

The plugin stores generated summaries in your WordPress database. No personal data is collected or transmitted to the plugin author.
